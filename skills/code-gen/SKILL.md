이슈 번호: $ARGUMENTS

위 이슈 번호에 대해 다음을 수행해줘:

1. `gh issue view $ARGUMENTS` 로 이슈 내용을 확인해줘.

2. 이슈에 작성된 작업계획을 분석해줘:
   - 어떤 파일을 생성/수정해야 하는지
   - 각 파일에서 어떤 변경이 필요한지

3. 작업계획에 따라 코드를 작성해줘:
   - 기존 코드 패턴과 컨벤션을 따를 것
   - 필요한 테스트가 있으면 함께 작성
   - 변경사항을 설명하면서 진행

4. 완료 후 변경된 파일 목록을 요약해줘.
